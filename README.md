# rtmp
-----------------------------------------------------------------------
library文件夹里面,Rtmp文件夹是这个demo使用的library,这个library是根据yasea-master的项目和在网上找到的so文件整理来的,原作者编译的so文件在p89(mini)设备上会报错;
yasea-master文件夹是https://github.com/begeekmyfriend/yasea的代码以及在网上找到的so文件。

-----------------------------------
示例工程,传入正确的推流地址,可以进行音频和视频的推流直播;

可以通过http://daniulive.com:8080/files/SmartPlayer/SmartPlayer.html进行推流音频和视频的验证。
